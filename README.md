# SitoWallet

SitoWallet is a personal finance mobile app for checking daily balance, tracking income and expenses, managing accounts, and keeping subscriptions visible.

## Potential Features

- Dashboard with current balance, weekly spending, monthly income, and upcoming renewals
- Account overview with balances and last sync state
- Recent transaction feed with categories and account context
- Subscription reminders and basic category shortcuts
- Offline-first style mock data ready to be replaced by API-backed state
